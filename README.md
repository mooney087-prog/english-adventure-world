# English Adventure World — 5 Levels

เกมสำรวจและสะสมคำศัพท์ 5 ด่าน รวม 25 คำ

## ด่าน
1. Colors: red, blue, yellow, green, pink
2. Classroom Objects: book, pencil, chair, bag, ruler
3. Kitchen: spoon, fork, plate, cup, pan
4. Places in School: classroom, library, canteen, playground, computer room
5. Places in the City: hospital, bank, market, park, police station

ทุกคำมีคำแปลภาษาไทย เสียงคำสั่ง เช่น “Pick up red.” และเสียงเอฟเฟกต์เมื่อเก็บคำศัพท์

## วิธีอัปโหลด
อัปโหลด `index.html`, `style.css`, `app.js` และโฟลเดอร์ `assets` ขึ้น GitHub Pages

## การปรับปรุงเวอร์ชันนี้
- ลบกรอบเรืองแสงของคำตอบทั้งหมด
- นักเรียนต้องฟังคำสั่ง วิเคราะห์ และค้นหาคำศัพท์เอง
- เก็บผิดชิ้น ระบบจะบอกให้ลองใหม่โดยไม่เฉลยตำแหน่ง
